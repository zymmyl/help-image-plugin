# 帮助图片插件 (help-image-plugin)

上传、管理和发送帮助图片，支持合并转发、图片备注、定时推送、机器人与群聊权限控制。

> 最后更新：2026-07-16

## 指令列表

| 指令 | 权限 | 说明 |
|------|------|------|
| `#全部帮助` | 所有人 | 发送所有已保存的帮助图片 |
| `#帮助指令` | 所有人 | 查看本插件所有指令及用法 |
| `#上传帮助图` | 主人 | 引用图片后发送，保存到插件资源目录 |
| `#删除帮助图 序号` | 主人 | 按序号删除指定帮助图（如 `#删除帮助图 3`） |
| `#清空帮助图` | 主人 | 清空所有已保存的帮助图片 |
| `#帮助图数量` | 主人 | 查看当前帮助图片数量 |
| `#设置帮助图备注 序号 备注内容` | 主人 | 设置指定图片的备注文字 |
| `#删除帮助图备注 序号` | 主人 | 删除指定图片的备注 |

## 功能说明

### 上传帮助图

引用一条包含图片的消息，发送 `#上传帮助图`，插件会下载并保存引用消息中的所有图片到 `resources/images/` 目录。

### 全部帮助

发送 `#全部帮助`，插件会将所有已保存的图片发出：

- **1 张图片**：直接发送
- **2 张及以上**：合并转发发送，图片按文件名自然排序

合并转发中每张图片的标题格式为 `序号.备注内容`，例如：
```
1.主帮助面板，发送 #帮助 呼出
[图片]
2.星铁帮助面板，发送 *帮助 呼出
[图片]
```

如果配置了自定义文本，会在合并转发消息的最后追加一条文本消息。

### 帮助指令

发送 `#帮助指令`，插件会渲染一张精美的指令说明图片，展示所有指令及用法。

### 图片备注

每张帮助图都可以设置备注文字，合并转发时备注会显示在对应图片上方，格式为 `序号.备注内容`。

**设置备注：**
```
#设置帮助图备注 1 主帮助面板，发送 #帮助 呼出
```

**删除备注：**
```
#删除帮助图备注 1
```

### 删除帮助图

发送 `#删除帮助图 序号`，删除对应序号的图片。序号与 `#全部帮助` 合并转发中的排列顺序一致，从 1 开始。删除图片时会自动清理对应的备注。

### 定时推送

通过配置文件或锅巴界面设置定时推送，到时间后自动将所有帮助图合并转发到机器人所在的所有群聊（受群聊黑白名单过滤）。

## 配置项

配置文件位于 `config/config/config.yaml`，首次使用时从 `config/default_config/config.yaml` 自动复制。也可通过锅巴界面图形化配置。

### 合并转发设置

| 配置项 | 说明 |
|--------|------|
| `customText` | 合并转发消息底部追加的自定义文本，留空则不追加 |

### 机器人发送名单

| 配置项 | 说明 |
|--------|------|
| `botList` | 允许发送帮助图的机器人QQ号列表，留空则不限制 |

### 群聊黑白名单

| 配置项 | 说明 |
|--------|------|
| `groupWhitelist` | 群聊白名单，仅这些群可接收帮助图，留空则不限制 |
| `groupBlacklist` | 群聊黑名单，这些群不可接收帮助图，优先级高于白名单 |

### 定时推送设置

| 配置项 | 说明 |
|--------|------|
| `scheduledPush.enabled` | 是否启用定时推送 |
| `scheduledPush.cron` | Cron 表达式，如 `0 9 * * *` 表示每天9点 |

### 图片备注

备注存储在 `config/config/notes.json` 中，以图片文件名为 key。仅可通过指令进行管理。

## 配置示例

```yaml
scheduledPush:
  enabled: true
  cron: "0 9 * * *"

customText: "如有疑问请联系管理员"

botList:
  - 10000

groupBlacklist:
  - 111111111

groupWhitelist: []
```
